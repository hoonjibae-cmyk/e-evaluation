# e강의평가 v2.6.12

## 주요 변경
- 원장 내부 확인용 리포트의 `퇴원율 / 재원 안정성` 영역을 순위 막대 중심에서 `만족도 × 퇴원율 사분면 그래프`로 변경했습니다.
- 그래프 기준: 가로축은 만족도 낮음 → 높음, 세로축은 퇴원율 높음 → 낮음입니다. 오른쪽 위 1사분면은 만족도 높음 + 퇴원율 낮음인 안정권입니다.
- 기존 `관계 분석 r` 표기는 상관계수 설명이 직관적이지 않아, 리포트 화면에서는 분석 기준 문구로 교체했습니다.
- 강의평가 순위와 퇴원율 순위에서 같은 점수/같은 퇴원율은 같은 등수로 표시합니다.

## 재배포
```powershell
cd C:\e-evaluation-v0
npx.cmd vercel --prod --force
```


# e강의평가 v2.6.10

## 업데이트 요약

이번 버전은 선생님별 반 이름 매칭 방식과 원장 내부 확인용 리포트의 코멘트 화면을 정리한 핫픽스입니다.

### 1. 선생님별 반 이름 매칭 방식 선택 추가

`PDF/웹 리포트 생성` 화면의 개별 선생님 리포트 생성 영역에서 반 이름 매칭을 설정할 때 적용 방식을 선택할 수 있습니다.

- 양방향 적용: 정규반 ↔ 내신반처럼 두 반을 같은 그룹으로 합산할 때 사용합니다.
- 단방향 적용: 학기 변경에 따른 레벨 상승처럼 M3 → M4 형태로 이전반을 기준반으로만 합산할 때 사용합니다.

적용 범위는 기존과 동일하게 선택한 선생님의 모든 월 리포트이며, 다른 선생님에게는 적용되지 않습니다.

### 2. 원장 내부 확인용 리포트 코멘트 UI 정리

- `우선 확인 코멘트` 영역을 삭제했습니다.
- `선생님 좋은 점`, `선생님 아쉬운 점`은 선생님별 펼치기/닫기 카드로 표시합니다.
- 펼친 카드 안에서는 반복 문구를 제거하고 `학생 이름 · 반 · 학생 작성 코멘트`만 깔끔하게 표시합니다.
- 선생님 좋은 점/아쉬운 점은 별도 선생님별 영역에서만 확인하고, 하단의 기타 코멘트 목록에서는 중복 표시하지 않습니다.

### 3. 버전

- 앱 버전: `v2.6.10`
- 신규 SQL: 없음

## v2.6.10 수정 사항

- 원장 내부 확인용 웹 리포트 모바일 화면에서 표가 겹치거나 글자가 세로로 쪼개지는 문제를 개선했습니다.
- 선생님 전달용 웹 리포트의 반별 항목별 평균표도 모바일 카드형으로 표시되도록 개선했습니다.
- 모바일에서는 순위표/퇴원율표/학생별 응답 현황/평균표가 가로 표가 아니라 항목 라벨이 붙은 카드형 목록으로 표시됩니다.
- 트렌드 그래프, KPI, 코멘트 카드, 푸터가 모바일 폭에 맞게 1열 구조로 정리됩니다.

## v2.6.9 수정 사항

- 원장 내부 확인용 리포트의 학생별 응답 현황을 선생님별 펼치기/닫기 카드로 정리했습니다.
- 반별 점수 흐름 그래프의 표시 범위를 50~100점으로 조정했습니다. 50점 미만은 시각화상 50점으로 표시됩니다.
- 반별 점수 흐름 그래프 상단에 선택 선생님의 반 이름 매칭 범례를 표시합니다. 예: `이전반 → 기준반`, `정규반 ↔ 내신반`.
- 우선 확인 코멘트 영역은 표시하지 않습니다.

## v2.6.13 업데이트

- 원장 내부 확인용 리포트의 만족도 × 퇴원율 사분면 그래프 기준을 명확화했습니다.
- 만족도 기준선: 80점 이상 = 높음, 79점 이하 = 낮음.
- 퇴원율 기준선: 5% 이하 = 낮음, 5% 초과 = 높음.
- 그래프 내부에 X축/Y축 제목, 방향, 기준선 라벨을 표시합니다.
